# Portfólio — Bruno Aguiar

Portfólio II pessoal redesenhado com tema dark moderno, animações e responsividade total.

## 🚀 Acesse online
https://brunoaguiar1986.github.io/portifolio/

## ✨ Melhorias implementadas

- **Design moderno** — tema escuro com accent verde neon e azul ciano
- **Matrix rain** — efeito de fundo animado em canvas
- **Cursor customizado** — com efeito de seguimento suave
- **Navbar inteligente** — blur/glassmorphism, destaque da seção ativa
- **Hero section** — janela de código animada, estatísticas, botões CTA
- **Projetos** — cards com overlay de links em hover
- **Skills** — organizadas por categoria com hover effects
- **Contato** — links sociais + formulário estilizado
- **Animações** — scroll reveal em todos os elementos
- **Responsivo** — mobile, tablet, desktop
- **Hamburger menu** — navegação mobile suave
